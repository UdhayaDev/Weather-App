# Weather App 🌤️

A simple Flask-based web application to check real-time weather information for any city. The app shows the **temperature, weather description**, and a **weather icon**. The background color changes dynamically based on the weather/temperature.

---

## **Features**

- Built with **Flask** (Python backend)
- Responsive UI using **Bootstrap** (or TailwindCSS if you want)
- Displays:
  - City name
  - Temperature in Celsius
  - Weather description
  - Weather icon
- Dynamic background based on weather conditions
- Modern design with **centered layout** and **animated icons**
- JSON API integration with **OpenWeatherMap**
